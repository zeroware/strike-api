# Strike Search API client

This is a simple node.js API client for the _Strike Search_ website (http://getstrike.net)

## Quickstart

```js
	var strike = require('strike-api');
	
	strike.info('B425907E5755031BDA4A8D1B6DCCACA97DA14C04');
```
